This module sends an e-mail on each give badge action by achievement 
badge manager.

Installation
-------------

 * Copy the whole "notification_badge_manager" directory to your 
   modules directory - "sites/all/modules" and
 * Enable the module from path "admin/modules".
 * Go to "admin/structure/achievement/notification-badge-manager" and save Configurations.


Usage
-----
   
 * Go to "admin/structure/achievement/notification-badge-manager". Here you will see the 
   Notification badge manager -
   "structure >> Openbadging Configurations >> Notification Badge Manager."
 * You can edit mail format with notification templates.
